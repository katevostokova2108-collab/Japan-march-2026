Основные маршруты

  • Аэропорт KIX → Киото (Nankai / JR)
  • Киото → Окаяма (синкансэн)
  • Окаяма → Курасики (местный поезд)
  • Курасики → Хиросима (синкансэн)
  • Хиросима → Миядзима (паром)
  • Хиросима → Беппу (синкансэн + limited express)
  • Беппу → Кумамото (поезд)
  • Кумамото → вулкан Асо (автобус/поезд)
  • Кумамото → Осака (синкансэн)
